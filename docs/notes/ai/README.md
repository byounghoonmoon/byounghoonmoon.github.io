## 주요 AI 모델 비교

| 모델 타입 | 특징 | 주요 용도 | 예시 |
|----------|------|-----------|------|
| CNN | 이미지 패턴 인식 | 컴퓨터 비전 | ResNet, VGG |
| RNN | 순차 데이터 처리 | 자연어 처리 | LSTM, GRU |
| Transformer | 어텐션 메커니즘 | 언어 모델 | GPT, BERT |
| GAN | 생성 모델 | 이미지 생성 | StyleGAN, DCGAN |

::: warning 주의사항
AI 모델 사용 시 편향성과 윤리적 고려사항을 항상 염두에 두어야 합니다.
:::
